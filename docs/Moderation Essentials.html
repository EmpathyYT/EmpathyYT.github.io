Moderation Essentials

This is a simple plugin that includes 3 main commands: /onduty, /addmod, /remmod and /offduty```.

/onduty info#

This command has no sub commands, it clears the inventory and applies invisibilty then switches the player to creative mode, with creative invetory restrictions, /gamemode survival and /gamemode spectator are usable without OP during duty.

/addmod info#

This command has one subcommand, it adds a player to the mods of the server.

Subcommands#

  • Player, Type: PlayerName

/remmod info#

This command has one subcommand, it removes a player from the mods of the server.

Subcommands#

  • Player, Type: PlayerName

/offduty info#

This command has no sub commands, it teleports the player back to the location where /onduty was used, returns the old inventory, removes all effects and switches the player back to survival.

Command Examples#

  • /addmod EmpathyYTYT
  • /onduty
